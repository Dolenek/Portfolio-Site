# Jakub Dolének - Portfolio Site

Personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Highlights
- Home page with section navigation (`hero`, `projects`, `skills`, `contact`).
- Dedicated `/about` route for experience and education timeline.
- English/Czech localization with query-parameter language sync.
- Theme toggle with persistence and system preference fallback.
- Route-aware SEO metadata and JSON-LD.

## Stack
- React 19
- TypeScript 5
- Vite 7
- Tailwind CSS 3
- Framer Motion
- react-router-dom 7
- react-i18next

## Frontend Local Setup
```bash
npm install
npm run dev
```

Runs on `http://localhost:5173`.

## Scripts (root)
- `npm run dev`
- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Docker Deployment
From `deploy/`:
```bash
docker compose up --build
```

Entrypoint: `http://localhost:8081`

## SEO and i18n Notes
- Default locale is English (`en`); Czech is `?lang=cs`.
- Canonical and hreflang tags are generated by `Seo.tsx`.
- Sitemap and robots are in `public/`.
